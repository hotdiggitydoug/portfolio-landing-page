# Doug West Portfolio

A minimal, clean portfolio website.

## Files

- `index.html` - The main webpage content
- `styles.css` - All styling and layout

## Local Development

To view the site locally, open a terminal and run:

```bash
python3 -m http.server 3000
```

Then open http://localhost:3000 in your browser.

## Deployment

This site is deployed on Vercel. Any changes pushed to the `main` branch will automatically deploy.
